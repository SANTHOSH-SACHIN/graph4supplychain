# Graph Convolutional LSTM (GConvLSTM)

**Dataset**: Chickenpox

GConvLSTM combines graph convolutional layers with LSTM to improve the prediction of time-series data on graph structures.

## Results on Chickenpox Dataset:

- **MSE**: 0.9646
- **MAE**: 0.6288
- **MAPE**: 554.9057%
- **RMSE**: 0.9821
- **R-squared**: 0.0420
